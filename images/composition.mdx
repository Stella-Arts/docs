Most weak images aren't weak because the model is bad. They're weak because the description didn't think about *composition*. This page is a checklist of compositional decisions to make consciously.

## 1. Pick a shot type

Every image is *some* type of shot. If you don't specify, the model picks for you, and often picks wrong.

| Shot | Description | Use for |
| --- | --- | --- |
| **Extreme close-up** | Tight on a single feature (eyes, hands) | Drama, detail, emotion |
| **Close-up** | Head and shoulders | Portraits, expressions |
| **Medium shot** | Waist up | Conversation, characters in context |
| **Wide shot** | Full figure plus environment | Action, place, scale |
| **Establishing shot** | The whole environment, no character | Setting the scene |
| **Bird's-eye view** | Top-down | Maps, layouts, dramatic openings |
| **Worm's-eye view** | Looking up from below | Power, heroism, scale |
| **Over-the-shoulder** | From behind a character | Conversation, POV |

State the shot first or near the start. `Medium shot of a tired detective at a desk` is much more controlled than `A tired detective at a desk`.

## 2. Pick a focal length / lens style

Optional but powerful for photographic styles.

| Lens | Effect |
| --- | --- |
| `wide-angle (24mm)` | Lots of environment, slight distortion, dramatic |
| `standard (50mm)` | Natural perspective, looks like the human eye |
| `portrait (85mm)` | Compressed background, flattering for faces |
| `telephoto (200mm)` | Strong background compression, isolates subject |
| `macro` | Extreme close-up, shallow focus |

`A medium portrait shot, 85mm lens, soft background blur` will reliably look like a real portrait photo.

## 3. Pick a depth of field

| Term | Effect |
| --- | --- |
| `shallow depth of field` | Subject sharp, background blurry, isolates the subject |
| `deep depth of field` | Everything in focus, useful for landscapes |
| `bokeh` | Specifically that out-of-focus background blur |

## 4. Pick lighting

Lighting changes the *mood* of an image more than any other variable. Pick one.

| Lighting | Mood |
| --- | --- |
| `Golden hour` | Warm, nostalgic, romantic |
| `Blue hour` | Cool, melancholy, atmospheric |
| `Harsh midday` | Stark, exposed, dramatic |
| `Overcast` | Soft, neutral, calm |
| `Candlelit` | Intimate, warm, mysterious |
| `Moonlit` | Cool, eerie, lonely |
| `Neon` | Urban, electric, saturated |
| `Backlit` | Silhouetted, mysterious, dramatic |
| `Rim light` | Outlined, cinematic |
| `Volumetric light` | Visible light rays, atmospheric |

## 5. Pick a color palette

Not every image needs an explicit palette, but for stylized characters it helps a lot.

- `Warm earth tones, terracotta, ochre, deep brown`
- `Cool jewel tones, emerald, sapphire, amethyst`
- `Pastel, soft pink, mint, cream`
- `High-contrast, pure black and saturated red`
- `Monochrome blue` / `monochrome sepia`

## 6. Pick a mood word

A single emotional adjective at the end of a description does real work:

- `Serene`, `tense`, `wistful`, `ominous`, `playful`, `triumphant`

A serene forest at dusk reads completely differently from an ominous forest at dusk. Same place, opposite feeling.

## Putting it all together

Here's the same character in three compositional configurations:

**Portrait, intimate:**

```
*Close-up portrait of a young librarian, soft warm lamplight from the left,
hands folded on an open book, eyes lowered, shallow depth of field, melancholy mood,
oil painting style.*
```

**Wide, atmospheric:**

```
*Wide shot of a vast library at night, towering shelves disappearing into shadow,
a single figure at a desk lit by a green-shaded lamp, volumetric light through a high window,
ominous mood, dark academia oil painting.*
```

**Action, dynamic:**

```
*Medium shot, low angle, of a librarian running between shelves, books flying from her arms,
warm chase lighting from a window, motion blur on her hair, urgent mood, painterly anime style.*
```

Same character, same vibe, three completely different images. That's what composition control gets you.

<Tip>
  When an image feels "off" but you can't say why, walk through the six items above. Usually you skipped one. Add it explicitly and try again.
</Tip>
