This section is a reference. If you're looking for a step-by-step walkthrough, start at [Quick Creation](/quickstart) instead.

A character is defined by a stack of fields. Most are optional, supplying more isn't always better, it depends on what kind of character you're building. A famous figure may need almost nothing; a custom game-master character may need every field filled.

## The fields

## What each field does

| Field | Required | Max length | Affects behavior? |
| --- | --- | --- | --- |
| Name | Yes | 20 | Strongly |
| Greeting | Yes (Quick) | 500 | Strongly |
| Avatar | No |, | Visually only |
| Short Description | No | 50 | Mildly |
| Long Description | No | 500 | Moderately |
| Categories | No | 3 tags | Discovery only |
| Visibility | Yes |, | Access only |
| Image Style | No | 100 | Visually |
| Definition | No | 32,000 | Strongly |

<Tip>
  Most characters never need a full Definition. A great Name + Greeting + Long Description will carry you a long way.
</Tip>
