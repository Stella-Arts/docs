**Optional.** 0–50 characters.

The Short Description is a title, not a paragraph. It appears under the name in listings and on the character card. Its job is to make someone curious enough to tap.

## Format

Two patterns work well:

- **A complete short sentence**, `I help you practice intermediate Spanish.`
- **A noun phrase, like a job title**, `Spanish Teacher`, `Retired Detective`, `Sentient Coffee Maker`

Either works. Pick what reads more naturally for your character.

## What to put here

The most useful Short Description disambiguates the name. A character called `Maria` could be anything, but `Maria, Spanish Teacher` instantly tells you what to expect.

## Examples

| Name | Short Description |
| --- | --- |
| Maria | I can help you practice intermediate Spanish. |
| Chippy | A squirrel. Definitely a real squirrel. |
| Marcus | Stoic philosopher, reluctant emperor. |
| BoardWizard | Recommends your next favorite board game. |

## What not to do

Don't repeat the name. `Maria, Maria is a Spanish teacher` wastes the slot.

Don't write a sales pitch. `The best Spanish teacher you'll ever meet!!!` reads like spam.

<Tip>
  Read your Short Description out loud. If it sounds like a real person introducing themselves, keep it. If it sounds like marketing copy, rewrite it.
</Tip>
