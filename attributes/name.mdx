**Required.** 3–20 characters. Letters, numbers, spaces, dashes, and underscores only.

## Formats

The name can be any of:

- A first name, `Sarah`, `Marcus`, `Ji-eun`
- A full name, `Albert Einstein`, `Marie Curie`
- A handle or title, `BoardWizard`, `TheDetective`, `WhoWouldWin`
- A descriptor, `Tired Coffee Maker`, `Stage Fright`

## How the name shapes behavior

The name does more than label the character. It's one of the strongest signals the system uses to figure out who this character is.

- **Famous names**, `Sherlock Holmes`, `Hermione Granger`, `Marcus Aurelius`, pull in a huge amount of context automatically. The system has seen them in books, articles, and conversations during training. With a famous name, you can often write almost nothing else and still get a recognizable character.
- **Ambiguous names**, `Mary`, `Alex`, `Sam`, give the system almost nothing to work with. The character will lean entirely on the Greeting and Description fields. That's fine, just plan for it.
- **Made-up names**, `Zorvath`, `Pip-3000`, let you start from scratch with no preconceptions, but you'll need to do all the defining work yourself.

## First and last names disambiguate

`Albert` is ambiguous. `Albert Einstein` is not. If you're going for a specific real or fictional figure, use the full name.

## Uniqueness

Names aren't unique across the platform. Two creators can both make a character named `Maria`. Your username plus the character name is what makes a specific character findable.
