**Optional.** 0–500 characters.

The Long Description is where you give the character a bit of backstory, personality, or context, but the secret is to write it *as the character*, not *about the character*.

## Write it in their voice

A description written *about* the character:

> Chair is a sentient piece of furniture. It has been in the same room for fifty years. It is grumpy.

A description written *as* the character:

> I'm Chair. I've been in this room for fifty years. I've held writers, criers, and one very nervous cat. Don't ask me to move, that's not my job. Sit, or don't.

The second version doesn't just tell the system *what* the character is. It also demonstrates *how* the character talks. That's much more useful for shaping responses.

## What to include

A few things worth covering, when they apply:

- **Traits and mannerisms**, speech patterns, quirks, what they care about
- **Backstory**, only the relevant bits, not their whole life
- **What they like to talk about**, this nudges the conversation toward those topics
- **What they don't talk about**, useful for keeping characters in their lane

## Examples

A game character:

> I'm WhoWouldWin. Pitch me two fighters, fictional, real, mythological, whatever, and I'll break down who wins and why. I take this very seriously. I will cite stats.

A helper character:

> Spanish tutor here. I'll keep things simple if you're new, and ramp up if you're not. I mix in English when you're stuck. *No* judgment if you mess up tenses.

A roleplay character:

> Detective Mara Cohen. Twenty years on the force, last five in homicide. I don't trust easy answers. If you're here, you've got a case for me, talk.

## Length

Don't pad. A tight three-sentence description outperforms a sprawling paragraph. Use the field if you've got something specific to say; skip it if you don't.
